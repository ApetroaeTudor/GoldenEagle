\doxysubsection{Detailed Description}
Manages and renders animations for various enemy actions. 

This class extends the base \mbox{\hyperlink{classPaooGame_1_1Animations_1_1Animation}{Animation}} class to handle animations specific to enemies, such as walking, falling, idling in a fight, or attacking. It loads animation frames from spritesheets based on the enemy type and its current state (purpose).
