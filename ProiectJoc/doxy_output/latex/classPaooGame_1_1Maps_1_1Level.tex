\doxysubsection{Detailed Description}
Abstract base class for game levels, managing tile data and providing collision detection utilities. 

This class handles the storage of visual and behavioral tile data for a game level. It loads this data from CSV files. Additionally, it provides a suite of static utility methods for performing tile-\/based collision checks (falling, wall, ceiling, ground properties) against a given \mbox{\hyperlink{}{Hitbox}}.

Tile behaviors are typically encoded as integers\+:
\begin{DoxyItemize}
\item 0\+: Lethal (e.\+g., water, lava) -\/ Causes death or falling.
\item 1\+: Lethal / Fall-\/through (e.\+g., pit) -\/ Causes death or falling.
\item 2\+: Solid (e.\+g., ground, wall) -\/ Blocks movement.
\item Other values\+: Air or custom behaviors. 
\end{DoxyItemize}
