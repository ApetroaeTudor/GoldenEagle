\doxysubsection{Detailed Description}
Custom exception to indicate an error during a value storage or retrieval operation. 

This exception is thrown when there\textquotesingle{}s a problem processing an operation related to storing or accessing a value, often in the context of a data buffer or data management system. It extends the base {\ttfamily Exception} class.
