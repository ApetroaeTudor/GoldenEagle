\doxysubsection{Detailed Description}
Represents a static, non-\/animated image. 

This class extends the base \mbox{\hyperlink{classPaooGame_1_1Animations_1_1Animation}{Animation}} class but is designed to display a single, unchanging image. It does not perform any animation updates or loading of multiple frames. It\textquotesingle{}s suitable for entities or elements that have a fixed visual representation.
