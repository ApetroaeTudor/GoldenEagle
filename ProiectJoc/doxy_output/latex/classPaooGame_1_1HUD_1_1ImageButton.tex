\doxysubsection{Detailed Description}
Represents a clickable button that displays an image. 

This class extends \mbox{\hyperlink{classPaooGame_1_1HUD_1_1HUD}{HUD}} and provides functionality for drawing a button with a specified image, detecting mouse hover, and handling click events. The button\textquotesingle{}s appearance (background color, border) changes when hovered over.
