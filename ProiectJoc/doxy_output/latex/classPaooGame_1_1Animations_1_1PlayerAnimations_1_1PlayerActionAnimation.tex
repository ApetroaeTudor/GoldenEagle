\doxysubsection{Detailed Description}
Manages and renders animations for various player actions. 

This class extends the base \mbox{\hyperlink{classPaooGame_1_1Animations_1_1Animation}{Animation}} class to handle animations specific to the player character, such as walking, jumping, attacking, etc. It loads animation frames from spritesheets based on the player\textquotesingle{}s current state (purpose).
