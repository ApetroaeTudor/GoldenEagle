\doxysubsection{Detailed Description}
Represents the game state for Level 2. 

This class manages all aspects of Level 2, building upon the \mbox{\hyperlink{classPaooGame_1_1States_1_1State}{State}} class. It includes the level map, hero interactions, enemies, NPCs, items (save points, boosters, floppy disks), camera control, UI elements (pause button, context HUD), and handles transitions to other states (\mbox{\hyperlink{classPaooGame_1_1States_1_1FightState}{Fight\+State}}, \mbox{\hyperlink{classPaooGame_1_1States_1_1Level3State}{Level3\+State}}, \mbox{\hyperlink{classPaooGame_1_1States_1_1DeathState}{Death\+State}}, \mbox{\hyperlink{classPaooGame_1_1States_1_1PauseMenuState}{Pause\+Menu\+State}}, \mbox{\hyperlink{classPaooGame_1_1States_1_1ShopState}{Shop\+State}}). It also implements saving and loading of level-\/specific progress.
