\doxysubsection{Detailed Description}
Implements the specific strategy for a basic skeleton enemy. 

This class defines the attributes and behaviors unique to a basic skeleton enemy, including its speed, hitbox dimensions, health, damage, and animations. It follows a singleton pattern to ensure only one instance exists.
