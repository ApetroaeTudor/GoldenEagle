\doxysubsection{Detailed Description}
Represents a specific type of item in the game that acts as a \char`\"{}booster\char`\"{}. 

This class extends the abstract \mbox{\hyperlink{classPaooGame_1_1Items_1_1Item}{Item}} class and defines a booster item. It initializes its properties such as name, sprite sheet path, hitbox, and a static animation, using constants defined in \mbox{\hyperlink{classPaooGame_1_1Config_1_1Constants}{Paoo\+Game.\+Config.\+Constants}}.
