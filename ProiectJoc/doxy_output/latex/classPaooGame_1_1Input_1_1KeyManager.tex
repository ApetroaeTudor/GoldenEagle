\doxysubsection{Detailed Description}
Manages keyboard input by tracking the current and previous state of keys. 

This class implements \mbox{\hyperlink{}{Key\+Listener}} to receive keyboard events. It stores the current state (pressed or not pressed) of all keys and the state from the previous update cycle. This allows for checking if a key is currently held down or if it has just been pressed in the current cycle.
