\doxysubsection{Detailed Description}
Represents a clickable \char`\"{}\+Attack\char`\"{} button. 

This class extends \mbox{\hyperlink{classPaooGame_1_1HUD_1_1HUD}{HUD}} and provides functionality for drawing an attack button, detecting mouse hover, and handling click events. The button\textquotesingle{}s appearance changes when hovered over.
