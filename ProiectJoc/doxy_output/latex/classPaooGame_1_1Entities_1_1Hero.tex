\doxysubsection{Detailed Description}
Represents the main player character in the game. 

The \mbox{\hyperlink{classPaooGame_1_1Entities_1_1Hero}{Hero}} class extends \mbox{\hyperlink{classPaooGame_1_1Entities_1_1Entity}{Entity}} and implements specific behaviors for the player, including input handling, movement (running, jumping, grappling), animations, health management, and interactions with the game world and data management systems.
