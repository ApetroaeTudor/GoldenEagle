\doxysubsection{Detailed Description}
Custom exception to indicate that a required data buffer is not yet initialized or populated. 

This exception is thrown when an operation attempts to access or use a data buffer (for example within a Data\+Manager) that has not been properly prepared, typically meaning data has not been loaded into it from a source like a database. It extends the base {\ttfamily Exception} class.
