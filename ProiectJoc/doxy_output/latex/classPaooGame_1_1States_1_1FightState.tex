\doxysubsection{Detailed Description}
Represents the turn-\/based combat state of the game. 

This state manages the interaction between the player (hero) and an enemy. It handles turns, player actions (attack, flee, block), enemy actions, animations, UI elements (health bars, buttons, popups), and transitions to other states (\mbox{\hyperlink{classPaooGame_1_1States_1_1DeathState}{Death\+State}}, Victory leading back to Play\+State). It features a fade-\/in effect upon entering and fade-\/out effects for transitions.

A blocking mechanic is implemented using a progress bar during the enemy\textquotesingle{}s turn. On the enemy turn the progress values will vary between 0 and 50.
