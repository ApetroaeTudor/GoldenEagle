\doxysubsection{Detailed Description}
Provides a centralized way to access shared game objects and utilities. 

This class acts as a \char`\"{}references\char`\"{} or \char`\"{}links\char`\"{} holder, allowing various game components to easily access core systems like the main \mbox{\hyperlink{classPaooGame_1_1Game}{Game}} object, input managers (keyboard, mouse), the player \mbox{\hyperlink{}{Hero}}, \mbox{\hyperlink{}{Tile\+Cache}}, enemy strategies, and data management signals. It helps to reduce direct dependencies between components and simplifies passing around commonly needed objects.
