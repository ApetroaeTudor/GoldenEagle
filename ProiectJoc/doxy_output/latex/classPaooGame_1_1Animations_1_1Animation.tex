\doxysubsection{Detailed Description}
Abstract base class for managing and rendering sprite animations. 

This class provides the foundational structure and common properties for all types of animations used in the game, such as player animations, enemy animations, item animations, and visual effects. It defines abstract methods for loading, updating, and painting animations, which must be implemented by concrete subclasses.
