\doxysubsection{Detailed Description}
Represents a zone that can trigger a message when an entity or a specified area interacts with it. 

This class defines a rectangular area in the game world. When a target intersects with this zone, it can trigger an associated message. The zone can be static or dynamically linked to an entity\textquotesingle{}s position.

Updates the bounds of the trigger zone if it is associated with an entity. 

If an entity is linked to this zone, its bounds are recalculated based on the entity\textquotesingle{}s current hitbox position and the predefined offsets. If no entity is associated, this method does nothing.
