\doxysubsection{Detailed Description}
Manages and displays contextual messages to the player based on their proximity to predefined trigger zones. 

This \mbox{\hyperlink{classPaooGame_1_1HUD_1_1HUD}{HUD}} element is associated with a \mbox{\hyperlink{}{Hero}} and contains a list of \mbox{\hyperlink{classPaooGame_1_1HUD_1_1MessageTriggerZone}{Message\+Trigger\+Zone}} objects. When the hero enters a trigger zone, a corresponding message is displayed on the screen for a set duration. The message is typically shown at a fixed position near the top of the game window.

Updates the state of the \mbox{\hyperlink{classPaooGame_1_1HUD_1_1ContextHUD}{Context\+HUD}}. 

This method performs the following actions\+:
\begin{DoxyEnumerate}
\item Updates the bounding boxes of all associated trigger zones, especially if they are linked to moving entities.
\item Checks if the hero\textquotesingle{}s hitbox intersects with any of the trigger zones.
\item If an intersection occurs, it sets the \char`\"{}current\+Message\char`\"{} to the message from the triggered zone, calculates a conceptual message position (though the drawing uses a fixed screen position), and resets the \char`\"{}message\+Timer\char`\"{}.
\item If no trigger is active or the timer for the current message has expired (implicitly, as timer reset happens on new trigger), the \char`\"{}current\+Message\char`\"{} might be cleared (or remain from the last trigger until a new one is found). The current implementation prioritizes the first active trigger found. 
\end{DoxyEnumerate}
