\doxysubsection{Detailed Description}
Manages and renders animations for static items, potentially with a full-\/screen display option. 

This class extends the base \mbox{\hyperlink{classPaooGame_1_1Animations_1_1Animation}{Animation}} class. While named \char`\"{}\+Static\+Item\+Animation\char`\"{}, its animation update logic suggests it can still animate (looping by default). It primarily loads animation frames from a specified spritesheet and provides a method to paint the animation scaled to fit the entire screen.
