\doxysubsection{Detailed Description}
Represents the main menu state of the game. 

This class extends the abstract \mbox{\hyperlink{classPaooGame_1_1States_1_1State}{State}} class and defines the behavior and rendering for the main menu. It includes buttons for starting a new game, continuing a saved game, and quitting the application. It also features an animated background.

\item \char`\"{}\+Quit\char`\"{}\+: Exits the application. This method overrides the \mbox{\hyperlink{classPaooGame_1_1States_1_1State_afafd048638af99f0f0b115ddec2fac1f}{State\#update()}} method.
