\doxysubsection{Detailed Description}
Custom exception to indicate that an attempted operation is not allowed. 

This exception is thrown when a part of the game logic attempts to perform an action or access a resource for which it does not have the necessary permissions or when an operation is restricted under certain conditions. It extends the base {\ttfamily Exception} class.
