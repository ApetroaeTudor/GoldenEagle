\doxysubsection{Detailed Description}
Implements \mbox{\hyperlink{}{Mouse\+Input\+Listener}} and \mbox{\hyperlink{}{Mouse\+Motion\+Listener}} to handle mouse input events. 

This class tracks mouse button presses, releases, and movement. It provides methods to get the current mouse coordinates, the number of times the mouse has been pressed since the last release, and a mechanism to detect a single click event.
