\doxysubsection{Detailed Description}
Implements the \mbox{\hyperlink{interfacePaooGame_1_1DatabaseManaging_1_1DataManager}{Data\+Manager}} interface to manage game data using an in-\/memory buffer and an SQLite database. 

This class provides a concrete way to save and load game state. It uses a Hash\+Map as an in-\/memory buffer for quick access to game variables and interacts with an SQLite database for persistent storage of game saves and high scores. It follows a Singleton pattern for instantiation.
