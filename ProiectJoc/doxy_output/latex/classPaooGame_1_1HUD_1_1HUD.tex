\doxysubsection{Detailed Description}
An abstract base class for Heads-\/\+Up Display (\mbox{\hyperlink{classPaooGame_1_1HUD_1_1HUD}{HUD}}) elements. 

This class provides a common structure for various \mbox{\hyperlink{classPaooGame_1_1HUD_1_1HUD}{HUD}} components that might be associated with a game \mbox{\hyperlink{}{Entity}}. Subclasses must implement the \char`\"{}draw\char`\"{} method to define how they are rendered on the screen.
