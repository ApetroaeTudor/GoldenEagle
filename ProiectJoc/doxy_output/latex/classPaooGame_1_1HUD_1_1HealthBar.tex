\doxysubsection{Detailed Description}
Represents a visual health bar for an entity in the game\textquotesingle{}s Heads-\/\+Up Display (\mbox{\hyperlink{classPaooGame_1_1HUD_1_1HUD}{HUD}}). 

This class extends \mbox{\hyperlink{classPaooGame_1_1HUD_1_1HUD}{HUD}} and is responsible for drawing a health bar that reflects the current health of an associated \mbox{\hyperlink{}{Entity}}. It features a smooth animation for health changes and customizable appearance (position, size, colors).
