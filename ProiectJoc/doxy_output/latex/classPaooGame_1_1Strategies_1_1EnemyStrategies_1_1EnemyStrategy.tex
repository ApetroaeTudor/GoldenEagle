\doxysubsection{Detailed Description}
Defines an abstract base class for different enemy AI and property strategies. 

This class provides a common structure for defining the characteristics and behaviors of various enemy types in the game. Concrete enemy strategies (e.\+g., \mbox{\hyperlink{classPaooGame_1_1Strategies_1_1EnemyStrategies_1_1BasicSkeletonEnemyStrategy}{Basic\+Skeleton\+Enemy\+Strategy}}) will extend this class to implement specific attributes like speed, health, damage, animations, and how they interact with the game world.
