\doxysubsection{Detailed Description}
Manages and renders animations for floating items in the game. 

This class extends the base \mbox{\hyperlink{classPaooGame_1_1Animations_1_1Animation}{Animation}} class to handle animations for items that typically float or have a continuous animation loop (for example pickups). It loads animation frames from a specified spritesheet.

Updates the current state of the animation. 

This method progresses the animation to the next frame based on the animation\+Speed. Since {\ttfamily play\+Once} is typically false for floating items, the animation will loop back to the first frame after reaching the last one. If {\ttfamily play\+Once} were true and the animation finished, it would stop updating.
