\doxysubsection{Detailed Description}
Represents an abstract base class for all items in the game. 

This class defines the common properties and behaviors that all game items should possess, such as position, name, associated sprite sheet path, hitbox for collision detection, and an animation. Concrete item types must extend this class and implement the abstract methods for drawing and updating their specific logic.
