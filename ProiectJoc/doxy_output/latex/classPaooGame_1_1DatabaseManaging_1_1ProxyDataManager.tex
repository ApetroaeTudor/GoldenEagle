\doxysubsection{Detailed Description}
Implements the \mbox{\hyperlink{interfacePaooGame_1_1DatabaseManaging_1_1DataManager}{Data\+Manager}} interface using a Proxy pattern to control access and add validation to a \mbox{\hyperlink{classPaooGame_1_1DatabaseManaging_1_1ConcreteDataManager}{Concrete\+Data\+Manager}}. 

This class acts as an intermediary (proxy) for a \mbox{\hyperlink{classPaooGame_1_1DatabaseManaging_1_1ConcreteDataManager}{Concrete\+Data\+Manager}} instance. It enforces access control (via the {\ttfamily access} boolean parameter), validates input keys and values before delegating operations to the concrete manager, and ensures the concrete manager is instantiated. This separation allows for additional logic (like security checks or validation) without modifying the core data management implementation.
