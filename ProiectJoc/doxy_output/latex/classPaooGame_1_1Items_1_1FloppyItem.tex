\doxysubsection{Detailed Description}
Represents a \char`\"{}floppy disk\char`\"{} item in the game, used for saving progress. 

This class extends the abstract \mbox{\hyperlink{classPaooGame_1_1Items_1_1Item}{Item}} class. It features a floating animation where the item moves up and down. It initializes its properties such as name, sprite sheet path, hitbox, and a specific \mbox{\hyperlink{}{Floating\+Item\+Animation}} using constants from \mbox{\hyperlink{classPaooGame_1_1Config_1_1Constants}{Paoo\+Game.\+Config.\+Constants}}.
