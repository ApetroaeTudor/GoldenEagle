\doxysubsection{Detailed Description}
Represents the pause menu state of the game. 

This class extends the abstract \mbox{\hyperlink{classPaooGame_1_1States_1_1State}{State}} class and defines the behavior and rendering for the pause menu. It appears as an overlay on top of the currently running game level, providing options to continue the game, return to the main menu, or quit the application.
