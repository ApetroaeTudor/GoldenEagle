\doxysubsection{Detailed Description}
Represents a customizable vertical bar that displays a value with a gradient fill. 

This class provides a visual representation of a value as a vertical bar. The bar fills from bottom to top with a configurable color gradient. It supports animation for value changes, a cap value beyond which the bar changes color (for example to gray), and customizable dimensions, position, and colors.
