\doxysubsection{Detailed Description}
Represents the game state for Level 3. 

This class extends the abstract \mbox{\hyperlink{classPaooGame_1_1States_1_1State}{State}} class and defines the specific behavior, entities, and rendering for the third level of the game. It manages the Level 3 map, camera, enemies, items, UI elements like the pause button, and game logic such as grappling, transitions, and save/load functionality.
