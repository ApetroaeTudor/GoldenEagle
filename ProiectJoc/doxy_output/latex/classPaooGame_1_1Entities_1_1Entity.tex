\doxysubsection{Detailed Description}
An abstract base class for all game entities (e.\+g., Player, \mbox{\hyperlink{classPaooGame_1_1Entities_1_1Enemy}{Enemy}}). 

This class provides common properties and functionalities for game entities, such as position, health, hitbox, and rendering. It defines abstract methods that subclasses must implement to define specific behaviors and appearances.
