Detailed Description

Manages and renders animations for static items, potentially with a full-screen display option.

This class extends the base Animation class. While named "StaticItemAnimation", its animation update logic suggests it can still animate (looping by default). It primarily loads animation frames from a specified spritesheet and provides a method to paint the animation scaled to fit the entire screen.

void PaooGame.Animations.ItemsAnimations.StaticItemAnimation.updateAnimation ( )
inline

Updates the current state of the animation.

This method progresses the animation to the next frame based on the animationSpeed. By default (playOnce is false), the animation will loop. If playOnce were true and the animation finished, it would stop updating.
