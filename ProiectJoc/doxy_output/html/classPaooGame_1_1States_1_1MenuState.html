Detailed Description

Represents the main menu state of the game.

This class extends the abstract State class and defines the behavior and rendering for the main menu. It includes buttons for starting a new game, continuing a saved game, and quitting the application. It also features an animated background.

◆ update()

void PaooGame.States.MenuState.update ( )
inline

Updates the logic for the MenuState.

This method is called repeatedly in the game loop. It handles:

  • Updating the background animation.
  • Checking for mouse clicks on the "New Game", "Continue", and "Quit" buttons.
  • "New Game": Resets the game buffer and loads the appropriate level based on saved state (defaulting to level 1 if no state).
  • "Continue": Loads the game state from the buffer and transitions to the saved level.
  • "Quit": Exits the application. This method overrides the State#update() method.
