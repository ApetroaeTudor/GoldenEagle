Detailed Description

Provides a centralized way to access shared game objects and utilities.

This class acts as a "references" or "links" holder, allowing various game components to easily access core systems like the main Game object, input managers (keyboard, mouse), the player Hero, TileCache, enemy strategies, and data management signals. It helps to reduce direct dependencies between components and simplifies passing around commonly needed objects.
