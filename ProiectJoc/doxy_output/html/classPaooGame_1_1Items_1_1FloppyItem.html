Detailed Description

Represents a "floppy disk" item in the game, used for saving progress.

This class extends the abstract Item class. It features a floating animation where the item moves up and down. It initializes its properties such as name, sprite sheet path, hitbox, and a specific FloatingItemAnimation using constants from PaooGame.Config.Constants.

Constructs a FloppyItem object.

Initializes the floppy disk item at a specified position. It sets the item's name, sprite sheet path, creates its hitbox, and loads its floating animation. The item will float vertically around its initial y-position.

Parameters
reflinkA RefLinks object providing access to game-wide resources and utilities.
xThe initial x-coordinate of the floppy item in the game world.
yThe initial y-coordinate of the floppy item in the game world, which also serves as the center of its floating motion.

Draws the floppy item on the screen and executes its floating movement.

This method uses the item's FloatingItemAnimation to render its current frame at the item's current position. It also calls executeFloppyFloating() to update the item's y-coordinate for the floating effect.
