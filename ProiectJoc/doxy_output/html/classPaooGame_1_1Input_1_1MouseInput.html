Detailed Description

Implements MouseInputListener and MouseMotionListener to handle mouse input events.

This class tracks mouse button presses, releases, and movement. It provides methods to get the current mouse coordinates, the number of times the mouse has been pressed since the last release, and a mechanism to detect a single click event.

◆ isOneClick()

boolean PaooGame.Input.MouseInput.isOneClick ( )
inline

Detects if a mouse click has occurred and ensures it's reported only once per press.

It returns true if a mouse button is currently pressed and this specific press has not yet been reported as a click by this method. After reporting true once, it will return false for subsequent calls during the same press, until the mouse button is released and pressed again.

Returns
True if a new click is detected (mouse is pressed and not yet reported), false otherwise.
