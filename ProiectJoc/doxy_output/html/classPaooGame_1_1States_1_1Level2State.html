Detailed Description

Represents the game state for Level 2.

This class manages all aspects of Level 2, building upon the State class. It includes the level map, hero interactions, enemies, NPCs, items (save points, boosters, floppy disks), camera control, UI elements (pause button, context HUD), and handles transitions to other states (FightState, Level3State, DeathState, PauseMenuState, ShopState). It also implements saving and loading of level-specific progress.
