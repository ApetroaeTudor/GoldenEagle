Detailed Description

Represents the game state for Level 1.

This class manages all aspects of Level 1, including the level map, hero, enemies, items (save points, floppy disks), camera, UI elements (pause button, context HUD), and transitions to other states (FightState, Level2State, DeathState, PauseMenuState). It handles game logic such as player movement, collision detection with items and enemies, pausing the game, and saving/loading level-specific progress.
