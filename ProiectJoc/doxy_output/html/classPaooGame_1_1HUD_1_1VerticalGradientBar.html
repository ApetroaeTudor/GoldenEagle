Detailed Description

Represents a customizable vertical bar that displays a value with a gradient fill.

This class provides a visual representation of a value as a vertical bar. The bar fills from bottom to top with a configurable color gradient. It supports animation for value changes, a cap value beyond which the bar changes color (for example to gray), and customizable dimensions, position, and colors.

void PaooGame.HUD.VerticalGradientBar.draw ( Graphics2D  g2d)
inline

Draws the vertical gradient bar on the screen.

This method handles the animation of the displayed value, draws the background, the gradient fill (or cap color if applicable), and the outline of the bar. The fill is a vertical linear gradient from startColor (bottom) through midColor to endColor (top).

Parameters
g2dThe Graphics2D context to draw on.
