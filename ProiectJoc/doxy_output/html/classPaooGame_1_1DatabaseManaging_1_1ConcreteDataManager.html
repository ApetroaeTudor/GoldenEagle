Detailed Description

Implements the DataManager interface to manage game data using an in-memory buffer and an SQLite database.

This class provides a concrete way to save and load game state. It uses a HashMap as an in-memory buffer for quick access to game variables and interacts with an SQLite database for persistent storage of game saves and high scores. It follows a Singleton pattern for instantiation.
