Detailed Description

Represents the third level of the game.

This class extends the abstract Level class and specifically defines the properties and data for Level 3. It initializes its tile data (both visual and behavioral) by loading from CSV files specified in PaooGame.Config.Constants.

PaooGame.Maps.Level3.Level3 ( )
inline

Constructs a Level3 object.

Initializes the dimensions for the visual tile array based on constants defined for Level 3 in PaooGame.Config.Constants. It then calls the setIDs(String, String) method (inherited from Level) to populate the visualIDs and behaviorIDs arrays by reading data from the CSV files specified by Constants#LEVEL3_TEXTURES_CSV and Constants#LEVEL3_BEHAVIOR_CSV. The behaviorTiles 2D array is not initialized in this constructor.
